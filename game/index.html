Concept

皆の心の支えとなる居場所

[皆の心の支えとなる居場所]がコンセプトのボードゲームサークルではメンバーの交流関係を広げる場であり、 皆のリラックスができる場所を提供することを目的としています。
色々


Servise

Event

新しい交流の場を作るボードゲームイベント

定期的にサークル外部の人を募集してボードゲームで遊びながら、 新しい交流関係を作る場を提供するイベントです。

Schedule

私生活を優先したラフな活動

バイトなどの私生活を優先しているので活動日などの融通がものすごくききやすく 活動時間も短めで気軽に参加できるようになっています

Create

新しいゲームを作る

時分だけの新しいゲームをサークルメンバーと協力することによって 更なる経験値を積み、より深い関係になることができる。
